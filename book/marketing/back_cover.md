# Back Cover Copy

Stop memorizing keystrokes. Start understanding the editor.

Most Vim books hand you a cheat sheet and wish you luck. VimFu hands you the model. It explains what modes really are, how the register system really works, and why the grammar of motions and operators is a tiny composable language — not a list of tricks to drill. Once you see the machine inside, the keystrokes stop being something to remember and become something you derive.

Every section of the book ends with a QR code that opens the matching collection of short, screen-recorded demos at vimfubook.com. The book stands on its own — you can read it cover to cover without ever picking up a phone. The companion website hosts a short screen-recorded video for every technique in the book and a browser-based Vim simulator where you can practice everything you read about, directly on the web page — no installation, no signup, no setup.

Whether you’re a Vim veteran who knows there’s a deeper layer, a VS Code or JetBrains user running a Vim plugin, or someone who’s bounced off the learning curve before — this is the book that finally makes the model click.
