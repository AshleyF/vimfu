# Amazon KDP Book Description

This is the description used in the KDP listing's "Book Description" field. KDP
accepts a limited subset of HTML (`<b>`, `<i>`, `<br>`, `<ul>`, `<li>`, `<h2>`)
and has a 4,000-character limit including tags. The version below comes in
around 1,300 characters.

Paste the contents of the fenced block below straight into the KDP description
field.

```html
<h2>Stop memorizing keystrokes. Start understanding the editor.</h2>

<b>VimFu</b> is a Vim &amp; Neovim book for programmers who are tired of cheat sheets that explain <i>what</i> the keys do but never <i>why</i>. Instead of reciting commands, it shows you the machine inside &mdash; the modes, the registers, the grammar of motions and operators &mdash; so the keystrokes stop being something to remember and start being something you <i>derive</i>.

<h2>What's inside</h2>
<ul>
<li><b>Foundations</b> &mdash; modes, the dot command, undo branches, and the mental model that makes everything else cheap.</li>
<li><b>The grammar of editing</b> &mdash; motions, operators, and text objects as a tiny composable language.</li>
<li><b>Hidden state</b> &mdash; registers, marks, macros, the jumplist, and what Vim is always tracking for you.</li>
<li><b>Search, substitution, and Ex</b> &mdash; the line editor still living one colon away.</li>
<li><b>Windows, buffers, tabs, and sessions</b> &mdash; what they actually are, not what they look like.</li>
<li><b>Modern Neovim</b> &mdash; LSP, Treesitter, plugins, and the ecosystem.</li>
</ul>

<h2>The book is the <i>why</i>. The site is the <i>show me</i>.</h2>

Every part ends with a QR code that opens the matching collection of short, screen-recorded demos at <b>vimfubook.com</b> &mdash; every technique in motion, at speed, in the real editor. Read the chapter. Scan the code. See it happen.

<br><br>

Whether you're a Vim veteran who senses there's a deeper layer (there is), a VS Code or JetBrains user running a Vim plugin, or someone who's bounced off the learning curve before &mdash; <b>VimFu</b> is the book that finally makes the model click.
```
