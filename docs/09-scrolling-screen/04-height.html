Set Window Height — z{n}Enter

An obscure but occasionally perfect resize trick.

Keys: z{n}{Enter}

z{n} followed by Enter resizes the current window to n lines tall and lands the cursor at the first non-blank of the topmost line.

z{n}Enter — Set Window Height

An obscure but occasionally perfect command: z10Enter resizes the current window to 10 lines and scrolls the cursor's line to the top.

Reference

Sequence Action
z{n}Enter Resize to {n} lines + scroll line to top
zEnter Cursor line to top + first non-blank
