z. / z+ / z- — Scroll and Land on First Non-Blank

Like zz/zt/zb, but the cursor jumps to the first non-blank.

id: scroll.first-non-blank-z · part: scroll · keys: z., z+, z- · lessons: 245, 246, 247

z. centers the line and moves the cursor to the first non-blank. z- puts the line at the bottom and lands at first non-blank. z+ scrolls forward and puts the next-page-top line at the top.

z. / z+ / z-

Three more z-family scroll commands. They behave like zz / zt / zb but also move the cursor to the first non-blank of the line — exactly where the code starts.

🔧 Under the Hood — Why Two Sets of z-Scrolls?

zz/zt/zb preserve the cursor's column. Useful if you want to keep the column for a follow-up motion. They came from the original vi.

z./z+/z- additionally move the cursor to the first non-blank. Almost always what you want when navigating code — landing in the middle of leading whitespace is rarely useful.

Mnemonic: the punctuation versions "clean up" the cursor position; the letter versions don't. Pick whichever you find ergonomic.

Tip. zEnter is a synonym for z+'s effect of "line below current top → new top, cursor first non-blank". The lessons cover z+; both keystrokes work.

Reference
