Horizontal Scroll

zs, ze, zh, zl — scroll left/right when wrap is off.

id: scroll.horizontal · part: scroll · keys: zs, ze, zh, zl, zH, zL · lessons: 289

When 'wrap' is off, long lines run off the screen. Horizontal scrolling commands let you see the rest. zs scrolls so the cursor is at the left edge; ze at the right.

Long single-line content (logs, JSON, CSV) runs off the right edge when 'wrap' is off. The z-family includes a horizontal scroll group that mirrors the vertical one.

🔧 Under the Hood — Only Useful When 'wrap' Is Off

By default, Vim wraps long lines (:set wrap). In wrap mode, horizontal scroll is meaningless — every character is visible somewhere on a continuation line.

Set :set nowrap for working with logs, CSV, JSON, etc. — then horizontal scroll matters. zs (start) and ze (end) reposition the viewport so the cursor sits at the left or right edge. zl/zh are one-column nudges; zL/zH are half-screen jumps.

'sidescroll' controls the step size for automatic horizontal scrolling when the cursor moves off-screen. 'sidescrolloff' is the horizontal counterpart to 'scrolloff'.

Reference

Key Action
zs Scroll cursor to left edge
ze Scroll cursor to right edge
zl Scroll one column right
zh Scroll one column left
zL Scroll half-screen right
zH Scroll half-screen left
