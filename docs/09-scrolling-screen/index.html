Scrolling Screen

  1. Half- and Full-Page Scrolling — Ctrl-D / Ctrl-U / Ctrl-F / Ctrl-B — scroll without losing place.
  2. Centering the Cursor — zz, zt, zb — move the viewport, not the cursor.
  3. z. / z+ / z- — Scroll and Land on First Non-Blank — Like zz/zt/zb, but the cursor jumps to the first non-blank.
  4. z{n}{Enter} — Set Window Height — Resize the current window to {n} lines, scroll cursor's line to the top.
  5. Line-by-Line Scroll — Ctrl-E and Ctrl-Y — scroll without moving the cursor (until you have to).
  6. Horizontal Scroll — zs, ze, zh, zl — scroll left/right when wrap is off.
