Windows Buffers Tabs

  1. Creating Splits — Ctrl-W s — horizontal split. Ctrl-W v — vertical.
  2. Navigating Windows — Ctrl-W h/j/k/l — move between splits. Ctrl-W c — close. Ctrl-W o — only.
  3. Move and Resize Windows — Ctrl-W H/J/K/L moves the window. Ctrl-W +/-/</> resizes.
  4. Buffers — Files in memory, navigated with :ls and :b.
  5. Tabs — :tabnew, gt, gT — tabs are layouts of windows, not files.
  6. Buffer vs Window vs Tab — Three concepts most editors collapse into one. Vim keeps them separate.
