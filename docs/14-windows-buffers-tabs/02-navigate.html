Navigating Windows

Ctrl-W h/j/k/l — move between splits. Ctrl-W c — close. Ctrl-W o — only.

Keys: Ctrl-W h, Ctrl-W j, Ctrl-W k, Ctrl-W l, Ctrl-W c, Ctrl-W o

Ctrl-W with a direction key moves to the next window in that direction. Ctrl-W c closes the current window. Ctrl-W o closes everything except the current window.

Ctrl-W is the window prefix. After it, h/j/k/l jump to the window in that direction. c closes; o keeps only this one.

Tip. Most users remap window navigation to plain Ctrl-h/Ctrl-j/Ctrl-k/Ctrl-l (no prefix). It's the single highest-impact remap for a multi-split workflow.
