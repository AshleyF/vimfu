Command Line Power

  1. Command-Line History — ↑/↓ for prefix-match, q: for the full window.
  2. Repeat Last Ex (@:) — @: replays your last :command. Then @@ repeats again.
  3. Editing the Command Line — Ctrl-B, Ctrl-E, Ctrl-W, Ctrl-U — readline-flavored.
  4. Paste from Buffer (Ctrl-R) — Same Ctrl-R as Insert mode — inject any register.
  5. Command-Line Completion — Tab to complete; Ctrl-D to list.
  6. The Wildmenu — 'wildmenu', 'wildmode', 'wildoptions' — making Tab at the : prompt actually pleasant.
