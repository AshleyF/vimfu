Paste from Buffer (Ctrl-R)

Same Ctrl-R as Insert mode — inject any register.

id: cmdline.paste-from-buffer · part: command-line-power · keys: Ctrl-R · lessons: 451, 451

From the command line, Ctrl-R{reg} inserts the register contents — exactly like in Insert mode. Lets you build commands from the unnamed register, search history, last yank, expression results, etc.

Paste from Buffer

Ctrl-R on the command line works exactly like in Insert mode: it injects register contents into the line you're building.

Sequence Inserts
Ctrl-R" Unnamed register
Ctrl-R0 Last yank
Ctrl-R+ System clipboard
Ctrl-R/ Last search pattern
Ctrl-R: Last Ex command
Ctrl-R% Current filename
Ctrl-RCtrl-W Word under cursor in buffer
Ctrl-R={expr} Expression result
