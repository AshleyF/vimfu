Sessions, Detach, Reattach

The feature that survives SSH drops and laptop reboots

id: tmux.sessions · part: tmux · keys: Ctrl-B d, tmux attach, tmux new -s, tmux ls · lessons: 524, 525, 526

Detach with Ctrl-B d, reattach with tmux attach, name sessions with tmux new -s name. Sessions outlive every terminal they run in.

A session is the outermost tmux container — a named bag of windows that keeps running even when you walk away. Detach with Ctrl-Bd: tmux closes its window, the session keeps running in the background, every shell inside stays alive. Come back hours or days later, type tmux attach (or tmux a for short), and you're back exactly where you left off.

Where Command
Inside tmux Ctrl-Bd — detach
Outside tmux tmux — attach to default unnamed session, or create one
Outside tmux tmux new -s work — start a new session named work
Outside tmux tmux attach — attach to last-used session
Outside tmux tmux attach -t work — attach to session named work
Outside tmux tmux ls — list all sessions
Outside tmux tmux kill-session -t work — kill named session
Inside tmux Ctrl-Bs — interactive session chooser
Inside tmux Ctrl-B: new-session — create from inside
Tip. If you SSH into a server, immediately run tmux (or tmux attach) before doing real work. SSH connections drop. tmux sessions don't. Anyone who has ever lost half an hour of work to a flaky network already knows why.

The Three-Layer Hierarchy

Sessions, windows, and panes nest:

Layer Examples in a typical setup Lives until
Session work, side-project, server You explicitly kill it
Window editor, server, logs (per session) You close it (Ctrl-B&) or session ends
Pane Editor + test runner side-by-side (per window) Its shell exits or you kill it (Ctrl-Bx)

The classic mistake is to use just one session for everything. Once you have more than five or six windows in one session, switching becomes a chore. Split into named sessions per project — work, personal, experiments — and use Ctrl-Bs or tmux switch-client to hop between them. Each session has its own window list, its own scrollback, its own state.
