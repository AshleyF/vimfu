Swap and Adjust Endpoints

o jumps to the other end. O switches corners in block mode.

Keys: o, O

While in visual mode, o swaps the cursor between the start and end of the selection — letting you adjust either endpoint without restarting. In visual-block mode, O swaps the diagonal corner.

Once in visual mode, the cursor sits at one end of the selection — and that's the only end you can extend. The o key swaps which end is active.

Tip. Mnemonic: lowercase o = opposite corner; capital O = Other side of the same row.
