Substitute (:s)

The most powerful single command in Vim.

id: ex.substitute · part: ex-commands · keys: :s, :%s · lessons: 406, 415

:s/pattern/replacement/flags is the workhorse for text transformation. Combine with ranges (% for whole file) and flags (g for global, c for confirm, i for case-insensitive).

Substitute

:s is Vim's find-and-replace. The full form is :[range]s/pattern/replacement/flags. Master it and you can rewrite a file in one keystroke.

Form Meaning
:s/foo/bar/ First match on current line
:s/foo/bar/g All matches on current line
:%s/foo/bar/g All matches in whole file
:%s/foo/bar/gc …with confirmation per match
:'<,'>s/foo/bar/g All matches in last visual selection
:5,10s/foo/bar/g Lines 5-10
:.,$s/foo/bar/g Cursor line to EOF
:g/TODO/s/foo/bar/g Only on lines matching TODO

Flags

Flag Effect
g Replace all matches on each line (not just first)
c Confirm each match (y/n/a/q/l)
i Case-insensitive for this command
I Case-sensitive for this command
e Errors suppressed (no "pattern not found")
n Don't substitute — just count matches
& Reuse flags from last :s

Replacement magic

In replacement Means
& The whole match
\0 Same as &
\1, \2… Capture groups from \(...\)
\u Uppercase next character
\U Uppercase until \E
\l, \L Lowercase versions
\r Newline (for splitting lines)
~ Reuse previous replacement
🔧 Under the Hood — What :s Actually Does

:s runs the regex engine over each line in [range] exactly once (unless g flag), substituting matches in place. The buffer's lines are rewritten line-by-line; the cursor lands on the last line changed.

  • Without [range], the range is the current line only (this trips up beginners — they expect file-wide and get one line).
  • The % range is shorthand for 1,$ (whole file).
  • The c flag enters an interactive prompt — y = yes, n = no, a = all (rest), q = quit, l = last (do this then quit).

Re-running: :& repeats the last substitute on the current line. :%& repeats on the whole file. & in Normal mode = same as :&.

Tip. :%s//bar/g reuses the last search pattern as the substitute pattern. Pair with * (search word under cursor) for a 5-keystroke replace-all.

Worked example — :s/foo/bar/g
