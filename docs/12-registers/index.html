Registers

  1. Registers — A dictionary of named clipboards built from one primitive
  2. The Unnamed Register — "" — where every yank and delete lands by default.
  3. Named Registers — "a–"z — your 26 personal slots. Capitals append.
  4. Numbered Registers — "0 = last yank. "1–"9 = the last nine deletes.
  5. The Small Delete Register — "- — for charwise deletes that wouldn't be worth a numbered slot.
  6. System Clipboard Registers — "+ and "* — bridge to the OS clipboard.
  7. Special Registers — "/ ": ". "% "# "= "_ — the read-only and weird ones.
  8. Macros Are Just Register Contents — "ap → edit → "ay$ — the macro lifecycle as text editing.
