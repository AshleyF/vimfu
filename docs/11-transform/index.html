Transform

  1. Replace Mode — R — overwrite as you type. r{c} — overwrite one character.
  2. Indent and Auto-Indent — >> indents. << dedents. ={motion} re-indents using the language indenter.
  3. Case Changing — ~ flips one character. gU/gu/g~ are operators.
  4. Joining Lines — J joins the next line to the current. gJ joins without a space.
