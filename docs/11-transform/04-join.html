Joining Lines

J joins the next line to the current. gJ joins without a space.

id: transform.join · part: transform · keys: J, gJ · lessons: 117, 118

J joins the line below to the current line, replacing leading whitespace with a single space (and removing comment leaders intelligently). gJ joins literally, with no whitespace fixup.

J joins the line below to the current line. By default it inserts a single space and trims leading whitespace from the joined line — almost always what you want for prose and code.

Tip. In Visual mode, J joins all selected lines. V-select-3-lines-J joins them all into one.
