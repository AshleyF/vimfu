Counts Visual

  1. Counts — Type a number before any motion or operator. It multiplies.
  2. Percentage and Column Jumps — {n}% jumps to a percentage of the file. {n}| jumps to a column.
  3. Character Visual Mode — v — select first, act second.
  4. Line and Block Visual — V selects whole lines. Ctrl-V selects rectangles.
  5. Visual Mode Tricks — Block O for diagonal corners. gv to re-select.
