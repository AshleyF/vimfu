Block Comment with Ctrl-V + I

Add a comment marker to {n} consecutive lines.

Keys: Ctrl-V, I, Esc

Ctrl-V to enter visual-block mode, j or count+j to select rows, I to insert at left edge, type the comment marker, Esc — the marker appears on every line.

Block Comment

Vim has no built-in "comment this block" command — but visual-block mode + capital-I gives you a one-trick replacement that works in any language.

Uncomment

Worked example — Ctrl-V jjj I//Esc

Block-prefix many lines at once.
