Patterns Recipes

  1. Swap and Duplicate Lines — ddp swaps two lines. yyp duplicates one.
  2. *, cw, then . . . — The Replace Loop — Search a word, change one, repeat with dot.
  3. cgn — Change-Next-Match Loop — Like *cw, but . jumps AND changes in one step.
  4. Block Comment with Ctrl-V + I — Add a comment marker to N consecutive lines.
  5. Increment and Decrement (Ctrl-A / Ctrl-X) — Find the next number, add or subtract — with counts.
  6. Walking the Numbered Registers — "1p, then u and . to walk the cut-history backwards.
  7. Diff Mode — vimdiff and :diffthis — visualize and edit changes side-by-side.
  8. External Filter Tricks — !{motion}cmd — let any Unix tool transform your text.
