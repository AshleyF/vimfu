Increment and Decrement (Ctrl-A / Ctrl-X)

Find the next number, add or subtract — with counts.

id: patterns.increment-decrement · part: patterns-recipes · keys: Ctrl-A, Ctrl-X · lessons: 468, 470

Ctrl-A increments the next number on the line; Ctrl-X decrements. With a count, it adds/subtracts that much. Combined with macros and :norm, generates sequences instantly.

Increment / Decrement

Ctrl-A adds 1 to the next number on the line. Ctrl-X subtracts 1. With a count: add/subtract that count.

Sequence Effect
Ctrl-A Increment next number on line by 1
10Ctrl-A Increment by 10
Ctrl-X Decrement next number
{count}Ctrl-X Decrement by count
{count}gCtrl-A Increment by count, with PROGRESSIVE step on visual selection
🔧 Under the Hood — g Ctrl-A and Sequences

On a visual selection of multiple lines, each containing a number:

  • Ctrl-A adds the count to every number.
  • gCtrl-A adds an increasing multiple of the count to each line in turn.

Classic recipe: write 100 lines of 0, select all, press gCtrl-A → you have 1, 2, 3, … 100.

Number recognition is controlled by 'nrformats' — by default decimal and hex (0x prefixed) and octal (0 prefixed; arguably surprising). Set :set nrformats-=octal to avoid the gotcha where 08 is treated as malformed octal.

Vim also recognizes negative signs and will respect them when wrapping (decrementing 0 gives -1).

Tip. Need a numbered list? 100i0<Esc> (insert 100 zeros) then Vgg (select all) then g<Ctrl-A> (= numbered 1..100).
