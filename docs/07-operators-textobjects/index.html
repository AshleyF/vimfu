Operators Textobjects

  1. Inner Word vs A Word — iw and aw — the first text objects.
  2. Quote Text Objects — i" / a" / i' / a' — operate on a quoted string without finding its boundaries.
  3. Bracket Text Objects — i( i{ i[ i< — operate on the contents of any bracket pair.
  4. Sentence and Paragraph — is / as / ip / ap — for prose.
  5. HTML/XML Tag Objects — it / at — operate inside an HTML or XML tag pair.
  6. The Vim Grammar — [count] [register] operator [count] {motion | text-object}
