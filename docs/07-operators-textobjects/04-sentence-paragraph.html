Sentence and Paragraph

is / as / ip / ap — for prose.

Keys: is, as, ip, ap

is is the sentence containing the cursor. ip is the paragraph. The 'a' variants include trailing whitespace.

Most of Vim's text objects are for code. These two are for prose — sentences and paragraphs as units of text.

Tip. gqip reformats the current paragraph to fit 'textwidth'. Combined with markdown-mode it's how many writers bulk-tidy a chapter.

Reference

Object Range
is Inner sentence (no trailing space)
as Sentence + trailing whitespace
ip Inner paragraph (no surrounding blank lines)
ap Paragraph + trailing blank line
