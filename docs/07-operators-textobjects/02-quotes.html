Quote Text Objects

i" / a" / i' / a' — operate on a quoted string without finding its boundaries.

Keys: i", a", i', a', i`, a`

i" is the contents of the surrounding double-quoted string. a" includes the quotes themselves. Same for ' and `.

Editing a string literal? Cursor anywhere inside it. ci" blanks the contents and drops you in to type the new ones. da" removes the quotes and contents together.

Tip. The three quote characters are ", ', and ` (backtick). All three behave identically — the object character matches the literal quote you typed. You don't need a separate object per language.
Tip. i" and a" are charwise — the cursor stays on the line. So da" doesn't delete the line; it deletes the quoted span and leaves the rest of the line intact.

Reference

Object Range
i" Contents of "…" (excluding quotes)
a" "…" including the quotes
i' / a' Same for '…'
i` / a` Same for …
