Counted Macros

100@a — run a macro a hundred times. Or until it errors out.

Keys: {n}@a

Counts work on macros. 100@a runs macro a up to 100 times; if any iteration aborts (search miss, end of file), playback stops cleanly.

A macro that does its work and moves to the next item is a loop. {n}@a runs that loop n times — but it stops as soon as the macro errors. So you can confidently say "do this 1000 times" knowing it'll stop when there's nothing left.

Tip. If you'd rather suppress errors and keep playing, use 100@a inside an :silent! or wrap with try/catch in scripting. For interactive work, the abort-on-error behavior is what you want.
