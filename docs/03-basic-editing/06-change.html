Change

c is delete-then-insert, atomically.

Keys: c, cw, C, cc, S, s

c is the change operator. cw changes a word; cc changes a line; C changes to end of line. S and s are short forms that change a whole line and a single character respectively.

c is the shortcut for d-then-i: it deletes the same range d would, and drops you straight into insert mode at that spot. Then Esc closes the change as one undoable, dot-repeatable unit.

Tip. Watch out: cw behaves like ce (change to end of word) — not what dw does. This is a deliberate, ancient quirk. The change operator with a word motion changes just the word, not the trailing whitespace. Most people prefer it that way.
Tip. Three handy shortcuts to remember: C is the same as c$ (change to end of line). S is the same as cc (change the whole line). s is the same as cl (substitute one character). s doesn't get a lot of love — it's a one-character substitute, useful when you want to type a replacement immediately and skip the explicit r-then-look-around.

Reference

Key Action Equivalent
c{motion} Change range d{motion} + i
cc Change line —
C Change to end of line c$
S Change line cc
s Substitute one char cl
cw Change word (note: like ce) —
ciw Change inner word (text object; see Text Objects) —

Worked example — cw is c then w
