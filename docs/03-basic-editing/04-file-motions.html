File Motions

gg, G, {n}G — jump anywhere in the file.

Keys: gg, G, {n}G, :{n}

gg goes to line 1. G goes to the last line. {n}G or :{n} goes to line n.

Three jumps cover most navigation across a file: top, bottom, and a specific line number.

Tip. 50G goes to line 50. So does :50Enter. Pick whichever your fingers reach faster.
Tip. gg and G are also operands. dG deletes from here to end of file (linewise). ygg yanks from here to top. =gg re-indents from cursor to top. Consistency, again.

Reference

Key Action
gg First line
G Last line
{n}G Line {n}
{n}gg Line {n} (alternate)
:{n}Enter Line {n} via Ex
{p}% Line at {p} percent of file

Worked example — gg G

Top and bottom of file.
