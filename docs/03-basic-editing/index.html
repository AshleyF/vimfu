Basic Editing

  1. More Ways into Insert — Five doors into Insert mode — pick the one that lands the cursor where you want.
  2. Word Motions — Move a word at a time. The grammar's first real noun.
  3. Line Motions — 0 / ^ / $ — the three places that matter on a line.
  4. File Motions — gg, G, {n}G — jump anywhere in the file.
  5. Delete — d is the verb. Pair it with any motion.
  6. Change — c is delete-then-insert, atomically.
  7. Yank and Put — Yank is copy. Put is paste. The register knows whether it's a line or a span.
  8. Replace a Character — r{char} — overwrite one character without entering Insert mode.
