Literal Insert and Digraphs

Ctrl-V for literals, Ctrl-K for digraphs — type the un-typeable.

Keys: Ctrl-V, Ctrl-Q, Ctrl-K

Ctrl-V (or Ctrl-Q on Windows) inserts the next keystroke literally — including special keys and Unicode codepoints. Ctrl-K combines two characters into one via the digraph table.

Sometimes you need to type a character that isn't on your keyboard, or you need to type a literal Tab that won't be expanded by autoindent. Ctrl-V and Ctrl-K are the escape hatches.

Literal Insert: Ctrl-V

Sequence Inserts
Ctrl-VTab Literal tab character (no shiftwidth math)
Ctrl-VEsc Literal Esc (^[) into the buffer
Ctrl-VEnter Literal CR (^M)
Ctrl-V065 Decimal codepoint 65 → A
Ctrl-Vx41 Hex codepoint 0x41 → A
Ctrl-Vu00e9 Unicode U+00E9 → é

Digraphs: Ctrl-K

Sequence Inserts
Ctrl-Ke' é (e-acute)
Ctrl-Kn? ñ
Ctrl-KOK ✓ (check mark)
Ctrl-K->| → (right arrow)
Ctrl-KPi π
