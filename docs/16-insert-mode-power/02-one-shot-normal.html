One-Shot Normal Mode

Ctrl-O runs one Normal command without leaving Insert.

id: insert.one-shot-normal · part: insert-mode-power · keys: Ctrl-O · lessons: 369, 370

Ctrl-O from Insert mode lets you fire one Normal-mode command and snap right back to insertion. The pinnacle of stay-in-flow editing.

Ctrl-O from Insert mode runs exactly one Normal-mode command, then drops you back into Insert at the new cursor position. It's the express elevator between modes.

From Insert Effect
Ctrl-Odiw Delete inner word, stay in Insert
Ctrl-Ozz Center the screen
Ctrl-O0 Jump to start of line
Ctrl-Op Paste at cursor
Ctrl-Ou Undo last Normal-mode edit
🔧 Under the Hood — Why It Counts as One Command

The Insert-mode Ctrl-O sets an internal flag (restart_edit) that says: finish exactly one Normal-mode operation, then resume Insert at the cursor. Operator + motion still counts as one — diw is one command. But . (dot-repeat) executed via Ctrl-O replays whatever the previous Normal command was, which can surprise you.

Note: any Insert-mode change you've already made up to this point is committed to the undo tree as a single edit. The next character you type starts a fresh undo unit.

Tip. Ctrl-O is what makes Vim's Insert mode bearable. Don't fight to stay in Insert — pop out for one command and come back.
