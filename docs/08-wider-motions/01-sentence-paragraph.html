Sentence and Paragraph Motions

( ) — sentences. { } — paragraphs.

id: wider-motions.sentence-paragraph · part: wider-motions · keys: (, ), {, } · lessons: 91, 92

( and ) jump to the previous and next sentence. { and } jump by paragraph (blank-line separated).

When you're navigating prose, words and lines are too small. Vim offers two larger units: sentences (delimited by ./!/?) and paragraphs (delimited by blank lines).

Tip. These pair with operators just like every other motion. d{key:}} deletes from the cursor to the next blank line. c)-type-replacement-Esc changes the rest of the current sentence.

Reference

Key Action
) Forward to next sentence
( Back to previous sentence
} Forward to next paragraph (blank line)
{ Back to previous paragraph
