Wider Motions

  1. Sentence and Paragraph Motions — ( ) — sentences. { } — paragraphs.
  2. Match Bracket — % — jump to the matching paren, brace, or bracket.
  3. Screen Position Jumps — H, M, L — High, Middle, Low on the visible screen.
  4. Line-Start Motions — +, -, Enter — jump to first non-blank of next/previous line.
  5. WORD Motions — W, B, E, gE — like w/b/e/ge but bigger words.
