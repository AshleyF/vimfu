Line-Start Motions

+, -, Enter — jump to first non-blank of next/previous line.

id: wider-motions.line-start · part: wider-motions · keys: +, -, Enter · lessons: 95

+ and Enter both move down to the first non-blank of the next line. - moves up. Useful for navigating indented code line-by-line without ending up mid-indent.

j moves down a line but keeps your column. So in indented code you often land mid-indentation. + (and Enter) move down a line and also land on the first non-blank — exactly where the code starts.

🔧 Under the Hood — These Are Linewise Motions to ^

Internally, + = j^ and - = k^. Enter is an alias for + for ergonomics — pressing Enter to advance to the next line is intuitive.

They're linewise motions, which means d+ deletes the current line and the next one (in linewise fashion), not just from cursor to next-line-start. Same with y+ — yanks two whole lines.

Reference

Key Action Type
+ / Enter Down to first non-blank Linewise
- Up to first non-blank Linewise
