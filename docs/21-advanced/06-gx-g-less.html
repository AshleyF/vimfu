gx and g<

Open URLs and recall command output.

Keys: gx, g<

gx opens the URL or filename under cursor in the system handler (browser/etc.). g< re-displays the output of the last :command — handy when a long message scrolled past.

Two small g commands worth knowing — both solve real problems.

gx — Open URL

gx on a URL or file path passes it to the system's default opener (open on macOS, xdg-open on Linux, cmd /c start on Windows). Browsers open URLs; PDFs open in PDF readers; image files open in image viewers.

g< — Re-show Last Output

g< reopens the message area showing the output of the last Ex command — handy when a :registers or :set all listing scrolled past faster than you could read.
