Visual Depth

  1. The Three Visual Modes — v, V, Ctrl-V — character, line, block.
  2. Block Insert and Append — I and A in block mode = insert/append on every selected line.
  3. Swap and Adjust Endpoints — o jumps to the other end. O switches corners in block mode.
  4. Operators on Visual Selections — Any operator works on a Visual range. Plus a few visual-only ones.
  5. Visual + Ex Range — Pressing : from Visual auto-fills the range.
  6. gv — Re-Select — Bring back the last visual selection.
