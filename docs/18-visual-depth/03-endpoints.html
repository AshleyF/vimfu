Swap and Adjust Endpoints

o jumps to the other end. O switches corners in block mode.

id: visual.endpoints · part: visual-depth · keys: o, O · lessons: 437, 438

While in Visual mode, o swaps the cursor between the start and end of the selection — letting you adjust either endpoint without restarting. In block mode, O swaps the diagonal corner.

Once in Visual mode, the cursor sits at one end of the selection — and that's the only end you can extend. The o key swaps which end is active.

🔧 Under the Hood — Why Two Anchors in Block Mode

A character or line Visual selection has only two endpoints (start and end), so o has just one place to send the cursor. A block Visual selection has four corners — top-left, top-right, bottom-left, bottom-right.

  • o swaps the cursor to the opposite corner (diagonal swap).
  • O swaps to the corner on the same row but other column (horizontal swap).

Between them you can land on any of the four corners and extend in any direction.

Tip. Mnemonic: lowercase o = opposite corner; capital O = Other side of the same row.
