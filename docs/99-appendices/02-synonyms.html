Synonyms and Shortcuts

Equivalent commands you'll see in different vimrcs.

id: appendix.synonyms · part: appendices

Many Vim commands have multiple keystroke spellings — D = d$, C = c$, S = cc, x = dl, X = dh, etc. Knowing the synonyms helps you read other people's code and your own old macros.

Vim has many single-key shortcuts that compress an operator+motion pair. Knowing them helps you type faster and read other people's keystrokes.

Shortcut Equivalent Meaning
D d$ Delete to end of line
C c$ Change to end of line
Y yy (NOT y$ — historical wart) Yank line
S cc Change whole line
s cl Substitute one char
x dl Delete char forward
X dh Delete char backward
& :s Repeat last substitute (current line)
ZZ :x Save if changed, quit
ZQ :q! Quit, discard changes
U (per-line undo, rarely used)
. (repeat last change)
& :& Repeat last :s
🔧 Under the Hood — The Y Wart

Logically Y should be y$ (yank to end of line) — that's the pattern set by D and C. But for historical reasons Y is yy (yank whole line).

Many users add nnoremap Y y$ to their vimrc to fix this. Neovim made this the default in 0.6+.
