Keyboard Walk

Every Normal-mode key, organized by physical key location.

id: appendix.keyboard-walk · part: appendices

A row-by-row tour of the QWERTY keyboard mapping each key to its primary Normal-mode function. Useful as a wall poster.

Vim's bindings cover the entire keyboard. This appendix walks the QWERTY layout row-by-row, showing the primary Normal-mode function of each key. (Many keys also serve as second halves of operator-motion sequences — see `appendix.complete-key-reference`.)

Capital letters (the un-shifted's siblings)

Most uppercase keys are bigger or opposite versions of their lowercase counterparts: • I / A — Insert at line start / Append at line end (vs. cursor position). • O — Open line above (vs. below). • E/W/B — WORD-flavored versions of e/w/b. • F/T — backward versions of f/t. • N — previous search match. • G — go to last line (vs. gg go to first).
