# ShellPilot Demos

This directory contains example demos built with ShellPilot.

## Running Demos

From the shellpilot directory:

```bash
python -m demos.vim_basics
```

Or directly:

```bash
python demos/vim_basics.py
```

## Available Demos

- **vim_basics.py** - Introduction to Vim: opening files, insert mode, saving and quitting
